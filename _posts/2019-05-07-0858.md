# C++ Style Cast

## static_cast
### Syntax
```
static_cast<typeYouWant>( expression )
```

## const_cast
### Syntax
```
const_cast<typeYouWant>( expression )
```

## reinterpret_cast
### Syntax
```
reinterpret_cast<typeYouWant>( expression )
```

## dynamic_cast
### Syntax
```
dynamic_cast<typeYouWant>( expression )
```
