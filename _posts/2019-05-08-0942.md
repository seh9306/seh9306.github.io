# git flow
```
git version 2.21.0.windows.1
```

## Initialize git flow
### Syntax
```
git flow init
```
master, develop branch가 생성된다.

## feature branch
### Syntax
```
git flow feature start [nameYouWant] [base]
git flow feature finish
```

## release branch
### Syntax
```
git flow release
```
